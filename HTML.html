HTML 문법 정리

0. HTML 문법의 기본 원리

HTML은 기본적으로 문법을 사용할 때, 여는 문법과 닫는 문법이 있다.
이를 구분하기 위해 여는 문법은 <문법 이름>, 닫는 문법은 </문법 이름> 로 표기한다.
그러나 예외적으로 <br>는 문장의 줄을 바꾸는 역할에 그치므로 닫는 문법이 없다.

1. 기본적으로 들어가야 하는 문법

<!doctype html>
<html>

<head>

    <meta charset = "utf-8">

    <title>적고 싶은 타이틀</title>

</head>

<body>

    <적용하고 싶은 기능들>

</body>

</html>

2. 특정 내용을 강조하는 문법

  1. <strong>내용</strong> - 글자를 진하게 만듦
  2. <u>내용</u> - 밑 줄을 그음

3. 글을 구분하는 문법

  1. <br> - 줄을 바꿈
  2. <p>내용</p> - 문단을 구분함
  3. <ol>내용</ol> - 리스트에 숫자가 매겨져서 나옴. Ordered list
  4. <ul>내용</ul> - 리스트에 숫자가 안매겨져서 나옴. Unordered list
  5. <li>내용</li> - ul및 ol을 사용할 때 써야하는 문법

4. 링크를 걸어주는 문법

<a href = "링크">
