CSS 문법 정리

0. CSS 문법 사용하는 법

html의 head 태그 안에 <style>과 </style>를 적은 뒤, 적용하고 싶은 태그를 적은 뒤, 중괄호 안에 적용하고 싶은 문법 작성. 이 때, 따로 태그가 없으면 <div>를 사용하면 됨. div 태그는 아무 기능도 없는 태그

1. 글자를 꾸미는 문법

  1. color : "바꾸고 싶은 색"; - 글자의 색을 바꿈.
  2. text-decoration:none; - 글자에 적용되어 있는 효과 삭제 ex) h1과 같은 태그를 사용할 때 붙는 밑줄 제거 가능.
  3. text-align: center, left, right, justify; - 글자를 중앙, 왼쪽, 오른쪽, 정렬해주는 기능

2. 박스 모델

  1. html에 적용되는 글자는 태그마다 고유의 속성을 가지고 있음. 이를 통해 box를 만들 수 있음. 바깥 쪽 부터 margin, border, padding이며, 이를 조절하며 시인성 좋게 만들기 가능
  2. border-top,bottom,left,right:아무 숫자px solid 색상; - 박스의 선을 상, 하, 좌, 우에 나타내며 두께는 px로 나타내고 색상도 설정 가능

3. 그리드 설정

  1. html에서 표를 만들고 싶으면 grid를 설정하면 된다. 이때 적용하는 방법은 disply:grid;라는 문법을 사용하면 된다.
  2. grid-template-columns: - 열을 생성하는 grid

4. 미디어 쿼리

  1. 화면의 size에 따라 어떻게 보일지 설정할 수 있는 문법이다.
  2. @media(적용하고 싶은 size)이후 중괄호 안에 style과 동일하게 적으면 됨.
